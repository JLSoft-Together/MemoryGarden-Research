# Copywriting Pack — MVP Flow

> Mục tiêu: gom copy ngắn cho UI để mockup/dev không tự viết lệch nhau.
> Nguồn chốt: `.planning/UX_REVIEW_MVP.md` + `.planning/MVP_SCREEN_SPEC.md`
> Trạng thái: planning only, **chưa code mockup**.

---

## 1. Brand tone

- Ấm
- nhẹ
- gợi cảm xúc
- không sến
- không game hóa quá đà
- ngắn, dễ đọc trên mobile

### Rule viết copy
- Ưu tiên câu ngắn
- 1 screen chỉ 1 thông điệp chính
- CTA dùng động từ rõ
- Locked tip phải kéo user quay lại core loop
