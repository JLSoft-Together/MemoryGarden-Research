# Memory Garden: Your Life Journey

## Overview

Memory Garden is a personal memory and life journey app that transforms real-life experiences into a growing digital garden.

Instead of tracking tasks, habits, or productivity goals, Memory Garden encourages users to capture meaningful moments, celebrate milestones, and watch their personal garden flourish over time.

- Every memory becomes a seed.
- Every experience helps the garden grow.
- Every journey creates a unique story.

---

## Core Features

### 🌱 Memory Capture

Create and save life memories in seconds.

Each memory can include:

- Title
- Description
- Photos
- Date
- Category
- Tags

Optional:

- Location
- Mood
- Favorite memory marker

Examples:

- First trip to Ha Long Bay
- Coffee date with a loved one
- Graduation day
- Watching a favorite concert
- Completing a personal goal

### 🌳 Living Garden System

Every saved memory plants a seed in the user's garden.

Over time, memories evolve through multiple growth stages:

- Seed
- Sprout
- Young Plant
- Blooming Flower
- Rare Plant

The more meaningful experiences users record, the more vibrant and unique their garden becomes.

### 🌸 Plant Collection

Discover and collect unique plants inspired by life experiences.

Examples:

- Rose of Love
- Coffee Blossom
- Sunset Lily
- Traveler Palm
- Wisdom Tree
- Sakura Memory

Players gradually unlock rare plants as they continue documenting their lives.

The collection is presented as an **Atlas/Pokédex**: a grid of plants, silhouette slots for locked species, a **dex number** per plant, and a **"catch" animation** when a new plant unlocks — turning empty slots into a collecting drive.

### 🚀 Onboarding & Activation

Get users to the "aha" moment fast.

- **"Plant a seed" onboarding**: welcome → pick style → plant your first memory → First Bloom → short tour (~5s to grasp the concept)
- **Gallery-seeded first memory**: pick one photo → EXIF + ML Kit auto-fills date/location/category → one tap creates the first memory
- **First Bloom Guarantee**: the first memory always blooms into a rare plant (season-immune)
- **3-Memory Starter Quest**: log the first 3 memories → unlock a badge
- **Quick-capture**: fast 1-photo + 1-line entry

### 📤 Social Share / Memory Card

Export a single memory or the whole garden as a **beautiful image card** (season palette + watermark) to share on social media — boosting brand awareness (ASO). Rendered on-device, no server.

### 🔒 Local-First Privacy

Memory Garden is designed with privacy in mind. User memories are stored locally on the device.

Benefits:

- Faster performance
- Offline support
- Better privacy
- No account required

Users maintain full ownership of their personal memories.

### ☁️ Backup & Export

Protect precious memories with backup options.

Features:

- Local backup
- Export to file
- Import from backup
- **Backup reminder**: nudge to Export ZIP when it has been a while — mitigates the data-loss risk of local-first
- **Manual Google Drive export** (optional): a copy the user actively creates, no auto-sync, no account
- Future cloud sync support
